Lists

The Lists​ prompt is: ​Create a project that uses list blocks.

In this module, students will explore lists. Lists can be found in the Variables category of the Block Palette. Lists can store multiple items of data (information such as strings or numbers). Students can make many different kinds of projects with lists. They could store user input to do a mad-libs-style project, or they could store the scores from each level that a user has played in a game.

This module is available as Google Slides. With the adjacent buttons, you can preview the module, edit the module by making a copy of the slides to remix and adapt for your teaching context, or download the module as a PDF.
